📦 部署步骤（3分钟完成）：

1️⃣ 打开 Netlify
   → 访问 https://app.netlify.com/drop

2️⃣ 拖拽上传
   → 把 "netlify-deploy" 文件夹直接拖到网页上

3️⃣ 完成！获得网址
   → 自动生成类似 xxx-12345.netlify.app 的网址

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔄 以后更新网站：

方法A - 简单更新（不保留历史）：
  → 重新拖拽文件夹到 https://app.netlify.com/drop

方法B - 保留版本历史（推荐）：
  1. 注册 Netlify 账号（用 GitHub 登录）
  2. 创建一个新 Site
  3. 连接到这个文件夹
  4. 以后更新这个文件夹后，Netlify 会自动部署
