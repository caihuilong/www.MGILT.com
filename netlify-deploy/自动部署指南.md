# GitHub + Netlify 自动部署指南

## 第一步：创建 GitHub 仓库

### 1.1 注册/登录 GitHub
1. 访问 https://github.com
2. 点击 "Sign up" 注册（用邮箱即可，免费）
3. 登录后完成

### 1.2 创建新仓库
1. 点击右上角 **"+"** → **"New repository"**
2. 填写：
   - **Repository name**: `mgilt-website` （仓库名称，可自定义）
   - **Description**: `MGILT 麦吉蓝图官网`
   - **Private/Public**: 选择 **Public**（公开）或 Private（私有）
   - ✅ **勾选** "Add a README file"
3. 点击 **"Create repository"**

### 1.3 上传网站文件
1. 在新仓库页面，点击 **"uploading an existing file"**
2. 把 `netlify-deploy` 文件夹里的所有文件拖进去
3. 点击 **"Commit changes"**

---

## 第二步：连接 Netlify

### 2.1 注册 Netlify
1. 访问 https://app.netlify.com
2. 点击 **"Sign up"**
3. 选择 **"Continue with GitHub"** （最方便）

### 2.2 添加新 Site
1. 登录后，点击 **"Add new site"**
2. 选择 **"Import an existing project"**
3. 点击 **GitHub** 图标
4. 第一次需要授权 GitHub 访问，点击 **"Authorize Netlify"**
5. 选择你刚创建的仓库 `mgilt-website`

### 2.3 配置部署设置
在配置页面：
- **Build command**: 留空（我们用纯HTML，不需要构建）
- **Publish directory**: `netlify-deploy` 或 `.` 或 `/`
- 点击 **"Deploy site"**

### 2.4 获得网址
等待1-2分钟部署完成，你会获得一个网址：
```
https://xxx-12345.netlify.app
```
（可以后续自定义域名）

## 第三步：以后如何更新

### 工作流程：
```
修改代码 → Push 到 GitHub → Netlify 自动部署
```

### 具体步骤：

1. **修改代码**
   - 直接在 GitHub 网页上编辑文件
   - 或下载到本地修改后再上传

2. **推送到 GitHub**
   ```bash
   # 在本地打开命令提示符，cd到网站文件夹
   cd G:\trae\huwaimatong\MGILT_Website

   # 初始化 Git（如果还没初始化）
   git init
   git add .
   git commit -m "更新网站"

   # 添加远程仓库（只做一次）
   git remote add origin https://github.com/你的用户名/mgilt-website.git

   # 推送
   git push origin main
   ```

3. **自动部署**
   - Netlify 检测到 GitHub 有新代码
   - **自动开始部署**（1-2分钟）
   - **自动更新网址**

## 额外功能

### 🔗 绑定自定义域名（可选）
1. 在 Netlify Site 设置中点击 **"Domain management"**
2. 点击 **"Add custom domain"**
3. 输入你的域名（如 `www.mysite.com`）
4. 按提示在域名服务商处添加 DNS 记录

### 📊 查看部署历史
- 在 Netlify 控制台可以看到每次部署的记录
- 可以回滚到之前的版本

### 🔔 设置通知
- 可以设置邮件通知，每次部署完成时收到邮件

## 常见问题

**Q: 部署失败了怎么办？**
A: 在 Netlify 查看部署日志，常见问题是路径配置错误。

**Q: 可以私有仓库吗？**
A: 可以，Netlify 免费版支持私有仓库。

**Q: 网址能改吗？**
A: 可以改 Netlify 子域名，在 Site settings 里修改。
